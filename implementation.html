Implementation

How to setup and run our project

1. Gather the Dataset

The first step is to download the codebase folder as a zip found here. You're free to clone this entire git repo, but if you just want a single directory, you can use this website to download a specific folder. Just paste in the github link to the directory.

Once you've downloaded the codebase folder, you have a few options. There is a mathematica notebook that can generate the graffitied images when run in the same directory as the unzipped base folder. However, I highly recommend against installing and running Mathematica (unless you really want a challenge). Instead, there is a zip file called images.zip that contains two folders, negatives and positives which have 400 examples each. The negatives do not contain any graffiti while the positive samples do. This images folder is all you need to train and run our model.

2. Run the Model

Once you've either unzipped the images folder or generated it through Mathematica, you're ready to run our model. Simply open up cnn-classifier.ipynb and run it. This will train the model based on the dataset and then display the results within the Jupyter notebook, like so:
